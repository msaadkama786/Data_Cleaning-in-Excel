# Data_Cleaning-in-Excel

Data cleaning in Excel is the process of identifying and correcting errors, inconsistencies, and duplicate data to improve data quality. Common data cleaning techniques include:

## Remove Duplicates
Select the data.
Go to Data → Remove Duplicates.
Choose the columns to check for duplicates.

## Trim Extra Spaces
Use the formula:
=TRIM(A2)
Removes leading, trailing, and extra spaces between words.
Change Text Case

## Convert text to uppercase:

=UPPER(A2)
Convert to lowercase:
=LOWER(A2)

## Convert to proper case:

=PROPER(A2)
Find and Replace
Press Ctrl + H.
Replace incorrect values, symbols, or text with the correct ones.

## Remove Blank Cells
Select the data.
Go to Home → Find & Select → Go To Special → Blanks.
Delete or fill blank cells as needed.

## Split Text into Columns
Go to Data → Text to Columns.
Split data using delimiters like commas, spaces, or tabs.

## Remove Non-Printable Characters
Use:
=CLEAN(A2)
Standardize Date Formats

## Select the date column.
Right-click → Format Cells → Date and choose a consistent format.
Data Validation
Go to Data → Data Validation.
Restrict entries to valid values, dates, or numbers.
Filter and Sort
Use Data → Filter to identify missing values, errors, or outliers.
